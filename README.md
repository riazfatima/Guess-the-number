# Guess the Number Game

**Created by: Riaz Fatima**

A simple Python game where you guess a random number between 1 and 100.

## How to Play

1. Run the program
2. Enter your guess (number between 1-100)
3. Get hints: "too small" or "too big"
4. Keep guessing until you find the correct number
5. Type `Q` to quit anytime

## Requirements

- Python 3.x
- No additional libraries needed (uses only built-in `random` module)

## How to Run

```bash
python "MP1 Guess the number .py"
